# To-Do List (CLI)

## Overview
This project is a command-line task management application built using Python. It provides a simple menu-driven interface for organizing daily tasks.

## Features
- Add new tasks
- View all saved tasks
- Delete existing tasks
- Menu-driven interface
- Input validation
- User-friendly interaction

## Technologies Used
- Python 3

## Project Structure
- task3.py

## How to Run
1. Download or clone the repository.
2. Open the project in VS Code.
3. Run the following command:

```bash
python task3.py
```
